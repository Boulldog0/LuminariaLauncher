Paramètres de votre RAM
❗ Important : Allouer beaucoup de RAM a votre jeu est essentiel. En effet, celui si ne pourra pas fonctionner correctement avec moins de 3 GO de ram alloué.
Définition de la RAM alloué : La ram alloué est la ram donné à votre jeu lors de son lancement. Une quantité plus important augmenteras les performances de votre jeu.
A noter qu'il faut éviter de mettre une trop grande quantité de RAM, sinon cela peut provoquer des pertes de performances sur vos autres applications.

- RAM Maximale : Indique la quantité totale de RAM allouée à votre jeu. Il est recommandé de ne pas dépasser 50% de la quantité totale de RAM présente sur votre PC afin qu'il puisse fonctionner correctement. Si votre ordinateur a plus de 8Go de RAM, nous vous recommandons d'allouer 3Go. Si votre ordinateur a 4Go ou moins, n'allouez pas plus de 2Go.

- RAM Minimale : Indique la quantité minimale de RAM allouée au démarrage de votre jeu. Il est recommandé de ne pas mettre plus de 3Go pour cette valeur. La valeur typique est de 1Go.

Mods Optionnels
Explications : Les mods présents dans cette fenêtre sont dits "optionnels", ce qui veut dire qu'ils ne sont pas obligatoire pour vous connecter au serveur et que vous pouvez choisir de lancer votre jeu avec ou non. Lorsque le curseur est sur vert, cela signifie que le mod sera bien lancé avec le jeu. Sinon, cela veut dire qu'il est désactivé et que ce mod ne sera pas lancé avec le reste de votre jeu. Si un mod est marqué comme "recommandé", cela signifie que les développeurs pensent qu'il est mieux pour votre experience de jeu que vous le possédiez.
Console
Paramètre après le lancement du jeu

Explications : Ce paramétre va définir ce que le launcher va faire une fois votre jeu lancé.
- Fermer le launcher après le lancement du jeu signifie que votre launcher sera fermé dés que le jeu se lance.
- Fermer le jeu si le launcher est fermé signifie que si vous fermez votre launcher, votre jeu se fermera immédiatement après.
- Garder le launcher ouvert après le lancement du jeu signifie que le launcher restera ouvert après le lancement de votre jeu.
